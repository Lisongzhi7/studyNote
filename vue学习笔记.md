## vue学习笔记

* 全局安装vue

```bash
npm install vue-cli -g
```

* 安装vue-cli

```bash
cnpm install vue-cli -g
```

* 创建项目

```bash
vue init webpack
```

* 项目运行

```bash
npm run dev
```
